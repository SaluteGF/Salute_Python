## 本地运行

1. 下载安装[Docsify](https://github.com/docsifyjs/docsify/)
2. 克隆[项目](https://github.com/SaluteGF/Salute_Python)到本地
3. cd进入项目根目录执行

```bash
docsify serve docs
```

4. 浏览器输入http://localhost:3000/#/ ，访问项目
